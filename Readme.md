# Secant method

Uso del método de Newton-Raphson para encontrar la raiz de una funcion, reemplazando la derivada de la funcion por:

```f'(x_n) = (f(x_n-1) - f(x_n)) / (x_n-1 - x_n)```

### [e^-x - x = 0](e^-x.sce)
```
Raíz de la función: e^-x - x = 0

Iteraciones:    23
x:              0.567143
```
